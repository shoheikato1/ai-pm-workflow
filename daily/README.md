# daily/

Daily notes generated by the `daily-summary` skill at end of day.

One file per day: `YYYY-MM-DD.md`

Each note includes: Shape of the Day, What Mattered (synthesized across meetings), Decisions Made, Open Threads, Actions Carried Forward, and an Energy and Momentum Check.

Not a meeting log. A synthesis — the things that will still matter in 3 months, extracted from everything that happened today.
