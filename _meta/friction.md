# Friction Log

Every time Claude loads the wrong context, produces output that needs correction, or something has to be re-explained from a prior session — log it here.

This is the bug tracker for the system itself. Entries reviewed during weekly system review. Each entry becomes a proposed fix to CLAUDE.md, a skill, or a context file.

---

## Format

```
### YYYY-MM-DD — [Short title]
**What happened:** [what went wrong or was slow]
**Root cause:** [why — wrong context loaded, missing skill, initiative confusion, etc.]
**Improvement vector:** vault structure / skill / CLAUDE.md rule / context file
**Proposed fix:** [specific change to make]
**Status:** OPEN / RESOLVED
```
